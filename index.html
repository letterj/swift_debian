Swift Lucid:

There is a debian package repository for Lucid at http://crashsite.github.com/swift_debian/lucid/ so you can easily install Swift on Lucid by:

  • Create /etc/apt/sources.list.d/swift.list:

    deb http://crashsite.github.com/swift_debian/lucid lucid main

  • Then:

    sudo apt-get update
    sudo apt-get install ...
