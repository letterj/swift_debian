Swift Lucid:

There is a debian package repository for Lucid at http://crashsite.github.com/swift_debian/lucid/ so you can easily install Swift on Lucid by:

  • Create /etc/apt/sources.list.d/swift.list:

    deb http://crashsite.github.com/swift_debian/lucid lucid main

  • Then:

    sudo apt-get update
    sudo apt-get install ...

The following packages are provided:

  • python-greenlet
    Needed by Eventlet.
  • python-eventlet
    Needed by Swift.
  • python-webob
    Needed by Swift.
  • python-swift
    The actual Swift code, needed by all other Swift packages.
  • swift
    General Swift tools, such as st and swift-ring-builder.
  • swift-proxy
    The Swift Proxy frontend service.
  • swift-account
    The Swift Account backend service.
  • swift-container
    The Swift Container backend service.
  • swift-object
    The Swift Object backend service.
  • swift-doc
    The Sphinx-generated documentation for Swift.
